ElementHider — Chrome Web Store 掲載用テキスト（日本語）


【拡張機能の紹介】

ElementHider は、Webページ上の不要な要素をワンクリックで非表示にできるChrome拡張機能です。DevToolsライクなピッカーで要素を選択するだけで、普段利用しないリンクやアイコン、メニューなど、目障りな要素を隠せます。


【インストールのメリット】

● ワンクリックで要素を非表示
DevToolsライクなピッカーで、消したい要素をクリックするだけ。CSSやHTMLの知識は不要です。

● プライバシー重視
必要なサイトにのみ個別に権限を許可する仕組みで、不要なアクセスを行いません。すべてのデータはブラウザ内に保存され、外部サーバーへの送信はありません。

● 設定はサイトごとに自動保存
一度非表示にした要素は、次回以降も自動的に隠されます。面倒な再設定は不要です。

● ちらつきのない非表示
ページ読み込みの最初期段階でスタイルを注入するため、非表示要素が一瞬表示されてしまう「ちらつき」がありません。

● 2つの非表示モード
要素を完全に消す「非表示」モードと、レイアウトを維持したまま見えなくする「透明」モードを要素ごとに選択可能。

● 6種のテーマ
ライト系・ダーク系を含む6つのテーマから、お好みの外観を選べます。

● 柔軟な管理機能
ドラッグ＆ドロップでの並べ替え、ラベルの編集、個別・一括の表示切り替えをサイドパネルから直感的に操作可能。

● データのエクスポート/インポート
すべてのサイトデータをJSONで書き出し・読み込みが可能。

● 日本語・英語対応
ブラウザの言語設定に応じて自動的に切り替わります。


=====================================
Chrome Web Store — プライバシー項目
=====================================


【単一用途の説明】

この拡張機能は、ユーザーが指定したWebページ上のHTML要素を非表示にする単一の目的で動作します。サイドパネルからDevToolsライクなピッカーを起動し、ユーザーがクリックした要素のCSSセレクタを保存して、該当要素を非表示にします。保存された設定は次回以降の訪問時にも自動的に適用されます。


【権限が必要な理由】

storage:
ユーザーが非表示に設定した要素のCSSセレクタやラベル情報、およびテーマなどのユーザー設定をchrome.storageに保存・読み込みするために必要です。すべてのデータはブラウザ内のローカルストレージに保存され、外部サーバーには一切送信されません。

sidePanel:
拡張機能のメインUIをChromeのサイドパネルとして表示し、表示・操作領域を確保するために必要です。ユーザーはサイドパネルから要素ピッカーの起動、非表示要素の一覧表示・管理（表示切り替え、削除、並べ替え、ラベル編集）を行います。

activeTab:
ユーザーがサイドパネルを操作した際に、現在アクティブなタブのURL（ホスト名）を取得するために必要です。これにより、サイトごとの非表示設定の保存・復元を行います。この権限はユーザーの操作時のみ有効で、バックグラウンドでのタブアクセスは行いません。

scripting:
ユーザーが権限を許可したサイトに対して、要素を非表示にするコンテンツスクリプト（CSSスタイルの注入と要素ピッカー）を動的に登録・実行するために必要です。スクリプトの登録はユーザーがサイトごとに明示的に権限を許可した場合にのみ行われます。
